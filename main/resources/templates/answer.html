3. 答錯
JavaScript 中哪個關鍵字用來宣告塊級作用域的變數？
A. var
B. let
C. const
D. function
查看解析
詳細解析
正確答案：B. let

解釋：

• let：ES6 引入的關鍵字，用來宣告塊級作用域的變數，不會發生變數提升問題。

• var：函數作用域，會發生變數提升，在現代 JavaScript 中較少使用。

• const：雖然也是塊級作用域，但用來宣告常數，一旦賦值後不能再修改。

• function：用來宣告函數，不是變數宣告關鍵字。
